# Portfolio Web 2026

Static Next.js portfolio focused on showcasing engineering projects, decisions, and measurable outcomes.

## Current status

- ✅ Documentation and planning complete.
- ✅ Initial Next.js + TypeScript + Tailwind scaffold implemented.
- ✅ Core routes created: `/`, `/projects`, `/projects/[slug]`, `/engineering`, `/resume`, `/contact`.

## Tech stack

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- Static export (`output: 'export'`)

## Getting started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build static export

```bash
npm run build
```
